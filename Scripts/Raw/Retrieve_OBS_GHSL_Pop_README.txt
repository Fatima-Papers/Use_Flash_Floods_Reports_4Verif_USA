The "Global Human Settlementc Layer - Population Grid (1km)" contains the absolute number of inhabitants of the cell (at 1km resolution), expressed as decimals (Float). 
The fields were retrieved at the following link:
https://ghsl.jrc.ec.europa.eu/download.php?ds=pop

The fields are saved as:
GHS_POP_E[year]_GLOBE_R2023A_54009_1000_V1_0.zip
in:
Data/Raw/OBS/GHSL_Pop/

Documentation on the database can be found here:
https://ghsl.jrc.ec.europa.eu/documents/GHSL_Data_Package_2023.pdf?t=1698413418

The database was accessed on 12/12/2023.
