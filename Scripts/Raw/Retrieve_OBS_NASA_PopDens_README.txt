1-km (30 sec) gridded population density dataset was chosen as predictor in this analysis.
The considered dataset was  the "Gridded Population of the World (GPW), v4.11" from the "Socioeconomic Data and Applications 
Center" (SEDAC, from NASA).

Documentation on the database can be found here:
https://sedac.ciesin.columbia.edu/data/set/gpw-v4-population-density-rev11

The geotiff files (one per year) were retrieved at the following link (free login required to download):
https://sedac.ciesin.columbia.edu/data/set/gpw-v4-population-density-rev11/data-download

The fields are saved in:

Data/Raw/OBS/NASA_PopDens/gpw-v4-population-density-rev11_2000_30_sec_tif.zip # valid between 2010 and 2015
